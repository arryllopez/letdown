# letdown

Inspired by *Oxenfree’s* 2.5D style and *Life is Strange’s* rich storytelling.

## Tech Stack

| Component | Tools / Libraries |
|----------|------------------|
| Core Language | C |
| Emulator (Testing) | PPSSPP |
| Development Kit | PSPSDK / pspsdev |
| Graphics API | sceGu |

## PSP-1000 Specifications

- **Display**
  - 4.3-inch TFT LCD
  - 480 × 272 pixels
  - 16.77 million colors

- **Processor**
  - PSP CPU (1–333 MHz)

- **Memory**
  - 32 MB Main Memory
  - 4 MB embedded DRAM
