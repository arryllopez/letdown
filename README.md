# letdown
undertale inspired indie game for the psp 

PSP-1000 (original):

CPU — MIPS R4000 @ 333MHz (often underclocked to 222MHz to save battery)
RAM — 32MB main RAM, 4MB VRAM
GPU — custom Sony GPU, hardware T&L, 33M polygons/sec
Storage — Memory Stick Pro Duo (up to 32GB), UMD disc
Screen — 480x272 LCD
Sound — hardware audio, stereo speakers

i am forcing myself to work within these constraints to create a full psp game
